1. Purchase and download the resource from [Tebex](https://ipl-mini-peds.tebex.io/).
2. Create a folder named `[mini-peds]` in your server's `resources` directory.
3. Extract all downloaded files into the `[mini-peds]` folder.
4. Open your `server.cfg` and add:
